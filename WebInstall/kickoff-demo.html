The <install> Element

A declarative HTML element for installing web apps

Install Current Page Same-Origin

Uses <install> with no attributes. The browser installs the current page as a web app.

<install>Install this app</install>

Setup & Instructions

Enable the feature

In the latest Edge/Chrome, enable the install element experiment:

  • Navigate to edge://flags (or chrome://flags)
  • Search for Web Install and enable the flag
  • Restart the browser

Or launch from the command line:

msedge --enable-features=WebInstall

Markup reference

Same-origin (installs the current page):

<install>Install this app</install>

Cross-origin (installs a different site):

<install installurl="https://example.com/">Install app</install>
