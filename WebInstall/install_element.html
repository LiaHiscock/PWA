Demo Instructions

In the latest version of your browser, ensure the Web Install API and the <install> element are enabled.

Alternatively, follow these instructions to enable experimental support:

  • Shut down the browser process.
  • Restart the browser with the appropriate command line flags, e.g.: 
    --enable-features=WebInstall
  • Ensure the flags actually took effect by verifying they appear on: 
    chrome://version

Markup Reference

Same-origin install (current page):

<install>Install this app</install>

Cross-origin install:

<install installurl="https://mustjab.github.io/">Install app</install>
